# Бегом во двор — лендинг

Маркетинговый лендинг проекта дворовых детских игр «Бегом во двор»
(инструктор приходит во двор, 45-минутные подвижные игры для детей 4–9 лет).

## Превью
GitHub Pages: https://xenaja.github.io/begom-vo-dvor/

## Структура
- `index.html` — лендинг (Фаза 1: вёрстка по спецификации, формы пока заглушки).
- `assets/` — изображения (логотип, иллюстрации детей, дудлы).
- `design_handoff_begom_vo_dvor/` — дизайн-хэндофф: спецификация (`README.md`) и референс.
- `visual-elements/` — исходные графические элементы.

## План
Лендинг + мини-CRM на Cloudflare (Pages + Worker + D1): расписание (локации/даты/места),
запись с оплатой через эквайринг Т-Банка, уведомления в Telegram и ВК, админка, бот-кабинет.
Деплой статики переедет на Cloudflare Pages; этот репозиторий — исходник.

Подробности дизайна — в `design_handoff_begom_vo_dvor/README.md`.
