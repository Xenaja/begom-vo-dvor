# Handoff: первый экран «Бегом во двор» — стиль обложки

Hero-блок (первый экран) лендинга проекта дворовых детских игр «Бегом во двор» в стиле фирменной
VK-обложки: зелёный фон, крупный рукописный вордмарк, подзаголовок с жёлтым акцентом, геометка
города и ряд белых дудл-иллюстраций снизу. Поверх — рабочая шапка-меню сайта.

Тон: светлый, тёплый, слегка игривый, «как в нашем детстве».

---

## О файлах

`reference/Бегом во двор — Hero обложка.html` — самодостаточный HTML-референс (все шрифты, стили
и изображения встроены; открыть в браузере двойным кликом — увидеть финальный вид и поведение).
Оригиналы изображений и шрифт — в `reference/assets/`.

> Это **дизайн-референс**, не продакшн-код. Воссоздайте экран в среде целевого проекта
> (React / Vue / Astro и т.п.) на нормальных компонентах и CSS — не переносите разметку дословно.
> Ниже всё, чтобы сделать это без исходника.

### Шрифты
- **Dudu** (Cyrillic, фирменный, Regular) — вордмарк, подзаголовок, геометка.
  Локальный файл `assets/dudu.ttf`, подключается через `@font-face` (семейство `Dudu`).
  Лицензия — свободная версия (vladmas). Запасной — любой рукописный/Comic-style.
  ```css
  @font-face{font-family:'Dudu';src:url('assets/dudu.ttf') format('truetype');font-weight:400;font-display:swap}
  ```
- **Nunito** (Google Fonts, OFL 1.1) — текст меню и кнопок. Импорт:
  ```
  https://fonts.googleapis.com/css2?family=Nunito:wght@400;700;800;900&display=swap
  ```

### Анимация
- `bobslow`: лёгкое всплывание `translateY(0 → -7px → 0)`, 7–10s, ease-in-out, infinite — на каждом дудле ряда (со слегка разной длительностью для «живости»).

## Структура экрана (сверху вниз)

### Шапка (sticky)
- `position: sticky; top:0; z-index:50`, фон `rgba(255,255,255,.92)` + `backdrop-filter: blur(10px)`,
  нижняя граница `1px #F0EEE6`. Контейнер `max-width:1180px`, padding `12px 24px`.
- Слева: круглый логотип `assets/logo.png` 50px (ссылка на `#top`).
- Справа (flex, gap 28px): ссылки-якоря «Как это работает» (`#how`), «Во что играем» (`#games`),
  «Вопросы» (`#faq`) + зелёная кнопка-пилюля «Записаться» (`#booking`).
- На ≤920px текстовые ссылки скрыты — остаются логотип и кнопка.

### Hero (зелёный фон `#9BC158`, overflow hidden)
1. **Геометка** в правом верхнем углу (`top:26px; right:28px`): жёлтый pin-SVG + «Санкт-Петербург» (Dudu, жёлтый).
2. **Контент по центру** (`max-width:1100px`, padding-top 88px):
   - Вордмарк **Бегом во двор** (Dudu 128px, белый). На широком экране ложится в одну/две строки —
     на узких уменьшается (см. адаптив).
   - Подзаголовок: «ДРУЖБА И ИГРЫ ВО ДВОРЕ,» (белый) + новая строка «КАК **В НАШЕМ ДЕТСТВЕ**»
     (последняя фраза жёлтая `#FBE27A`). Uppercase, letter-spacing 3px.
   - Кнопка-пилюля «Записать ребёнка» — белая, текст `#5E8C24`, ссылка на `#booking`,
     hover: приподнимается + усиливается тень.
3. **Ряд из 8 белых дудлов** снизу (`max-width:1180px`, flex, `justify-content:space-between`, `align-items:flex-end`):
   ребёнок с мячом · волейбол · ракетка · мяч · девочка со скакалкой · фрисби · волан · ребёнок с ракеткой.
   Дети крупнее (~84px), снаряды ~58–64px со сдвигом `margin-bottom:8px` для выравнивания по нижней линии.
   Все белые с прозрачным фоном, каждый с анимацией `bobslow` (разная длительность).

### Полоска-подсказка под hero (не часть обложки)
- Светло-зелёная плашка `#F6FAEE` с пояснением, что ниже идут блоки лендинга. На реальном сайте — убрать/заменить контентом.

## Адаптив
- **≤920px**: текстовое меню скрыто (логотип + кнопка); вордмарк → 74px; подзаголовок → 21px; дудлы ряда → 46px.
- **≤600px**: вордмарк → 52px; подзаголовок → 16px (letter-spacing 1px); ряд дудлов переносится (`flex-wrap`, по центру), дудлы → 38px; геометка → 18px.

## Что учесть при интеграции
- Якоря `#how`, `#games`, `#faq`, `#booking` ведут на блоки основного лендинга — связать с реальными секциями/страницей записи.
- Полоску-подсказку под hero на проде убрать.
- Вордмарк в одну строку требует широкого контейнера; на средних экранах допускается перенос — это нормально.
