# Handoff: блок «Поиграем?» (секция игр)

Промо-секция лендинга «Бегом во двор» со списком дворовых игр, раскиданных хаотично
по блоку — «как надписи на заборе». Зелёный фон, рукописные названия разного размера,
наклона и цвета, кнопка записи снизу.

Задача: воссоздать эту секцию в среде целевого проекта (React / Vue / Astro и т.п.) на нормальных
компонентах и CSS. `reference/games-block.html` — самодостаточный референс (открыть в браузере —
увидеть финальный вид и анимацию). Ниже — вся спецификация, чтобы собрать без исходника.

---

## Что за эффект

Названия игр выводятся из массива и раскидываются псевдослучайно, но **детерминированно**
(значения зависят от индекса, а не от `Math.random()`) — чтобы раскладка не «прыгала» при
перерисовке/перезагрузке. Каждому названию присваивается свой поворот, кегль, цвет (белый или
жёлтый), насыщенность и вертикальный сдвиг. Раскладка — обычный `flex-wrap` по центру, так что
секция остаётся адаптивной.

### Шрифты
- **Dudu** — фирменный рукописный, для **названий игр** (даёт «неровный», живой вид — без него читается хуже).
  Локальный файл `assets/dudu.ttf`, подключение через `@font-face` (семейство `Dudu`), запасной — Pangolin.
  ```css
  @font-face{font-family:'Dudu';src:url('assets/dudu.ttf') format('truetype');font-weight:400;font-display:swap}
  ```
- **Pangolin** (Google Fonts, OFL 1.1) — заголовок «Поиграем?» (округлый рукописный). Псевдо-жирность
  достигается через `-webkit-text-stroke: 0.7px currentColor` (у шрифта одно начертание).
- **Nunito** (Google Fonts, OFL 1.1) — подзаголовок и кнопка.

Импорт Google Fonts:
```
https://fonts.googleapis.com/css2?family=Pangolin&family=Nunito:wght@600;700;800;900&display=swap
```

## Структура секции (сверху вниз)

Контейнер: `max-width: 960px`, `padding: 74px 24px`, `text-align:center`, `position:relative`.
Секция `position:relative; overflow:hidden`.

1. **Декор фона** (абсолют, за контентом):
   - Дудл-мяч `assets/football.png` — `top:28px; right:-20px; width:140px; opacity:.18; rotate(8deg)`
   - Дудл-фрисби `assets/frisbee.png` — `bottom:24px; left:-24px; width:150px; opacity:.16; rotate(-10deg)`
   - Два скетч-облака (инлайновый SVG, белая обводка, полупрозрачные) с анимацией всплывания `bobslow`.
     Path облака: `M14 30 C6 30 4 22 11 20 C9 12 20 9 24 15 C27 8 40 9 41 17 C50 14 55 24 47 28 C46 30 44 30 42 30Z`, `viewBox="0 0 64 36"`.
2. **Заголовок** «Поиграем?» — Pangolin, 42px, белый, псевдо-жирный (text-stroke 0.7px), `margin-bottom:14px`.
3. **Подзаголовок** «Играем на свежем воздухе почти в любую погоду!» — Nunito 19px 700, цвет `#EFF7DF`, `margin-bottom:34px`.
4. **Облако названий** — контейнер `display:flex; flex-wrap:wrap; align-items:center; justify-content:center; gap:6px 30px; margin-bottom:52px`.
   Каждое название — `<span>` с индивидуальным стилем (см. алгоритм ниже).
5. **Кнопка** «Записаться на игры» — белая пилюля, текст `#5E8C24`, 19px/900, `padding:16px 40px`,
   `border-radius:999px`, ссылка на `#booking`. Hover: `translateY(-2px)` + усиление тени.

## Алгоритм раскидки названий

Для каждого названия по его индексу `i` (0-based):

```js
var rot    = ((i * 53) % 17) - 8;                       // поворот, -8..8 град
var size   = [30, 22, 38, 26, 34, 24, 28, 32][i % 8];   // кегль, px
var yellow = (i % 3 === 0);                              // каждый третий — жёлтый
var weight = [800, 700, 900][i % 3];                     // насыщенность
var dy     = [0, 10, -8, 6, -4, 12, -10, 4][i % 8];      // вертикальный сдвиг, px
```

Стиль одного `<span>`:
```
display:inline-block;
font-family:'Dudu','Pangolin',cursive;
font-weight:{weight};
-webkit-text-stroke:0.6px currentColor;   /* лёгкая доп. жирность */
font-size:{size}px;
line-height:1;
color:{yellow ? '#FBE27A' : '#ffffff'};
transform:rotate({rot}deg) translateY({dy}px);
white-space:nowrap;
```

Полный рабочий пример генерации — в `reference/games-block.html` (тег `<script>` внизу).

## Адаптив
Секция адаптивна за счёт `flex-wrap`. На узких экранах названия просто переносятся на большее
число строк. При необходимости на мобильном можно немного уменьшить базовые кегли (массив `size`).
